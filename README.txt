# Do-An-Quan-Ly-Ban-Hoa
Phan mem quan ly cua hang ban hoa
Thành viên:
1. Nguyễn Văn Huy
2. Phan Khánh Vinh
3. Tăng Cẩm Phú

Các bước để chạy project
1. Tại thư mục SQL nằm trong thư mục QuanLyCuaHangBanHoa ta chạy file QuanLyBanHoa.sql
2. Tại thư mục Application nằm trong thư mục QuanLyCuaHangBanHoa ta chạy tệp QuanLyBanHoa.sln
3. Tại thư mục DAO trong project, ta chọn DataProvider, sau đó điều chỉnh connectionString cho phù hợp với máy chủ
chứa cơ sở dữ liệu QuanLyBanHoa của project.
